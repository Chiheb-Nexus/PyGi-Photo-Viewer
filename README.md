**PyGI Photo-Viewer**is a simple photo viewer created with PyGI.

#Dependencies:
    ~ Python 3
    ~ PyGObject 3
    ~ Gtk3+
#Format supported:
    ~ JPG
    ~ JPEG
    ~ GIF
    ~ PNG
